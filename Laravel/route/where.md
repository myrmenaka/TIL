# 正規表現一覧（Laravelルートでよく使うもの）

| 正規表現 | 許可される文字 |  
|:--:|:--:|  
| [0-9]+ | 数字だけ |  
| [a-zA-Z]+ | 英字だけ（大文字・小文字） |  
| [a-z0-9]+ | 小文字英数字 |  
| [a-zA-Z0-9]+ | 英数字 |  
| [a-z0-9\-]+ | 小文字英数字＋ハイフン（slug定番） |  
| [a-zA-Z0-9\-]+ | 英数字＋ハイフン |  
| [a-zA-Z0-9_]+ | 英数字＋アンダースコア |  
| [0-9a-fA-F\-]+ | UUID（16進＋ハイフン） |  
| \d{4}-\d{2}-\d{2} | 日付（YYYY-MM-DD） |  
| .* | なんでもOK（キャッチオール） |
