## HTMLとは  

Hyper Text Markup Language  
`マークアップ言語`  
webページのコードを作るのがHTML  

## CSSとは  

Cascading Style Sheets  
`スタイルシート言語`  
見た目を装飾するもの  

`HTML ＝ 骨組`  
`CSS ＝ 肉付け（装飾）`  

webページの見える部分を作る言語  

HTMLの最新バージョンは　`5.1`  

CSSの最新バージョンは　`CSS3`  

### JavaScript  

- アニメーション  
- サーバーと連携してより高度な動きをつける  
- データのやり取り  

### サーバーサイド言語（PHP,Rudy）  

より幅広いことができる  

- ログインしてユーザー情報をデータベースに保存etc
